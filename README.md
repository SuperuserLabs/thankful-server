thankful-server
===============

A server that gathers information about creators online and serves that info through a REST API.

Information to collect:

 - User profiles
   - YouTube
   - Soundcloud
   - Patreon
   - etc.
 - Payment methods
   - Cryptocurrency addresses
 - Fanbase
   - Daily views
   - Subscribers
 - Donations
   - Using Patreon
   - Using crypto
